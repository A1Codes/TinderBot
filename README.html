Tinder Auto Liker Bot

This is a Python script that automates liking profiles on Tinder using Selenium web driver.

css Copy code

Installation

  1. Ensure that you have Python and pip installed on your machine.
  2. Install Selenium package by running the command "pip install selenium".
  3. Clone this repository.

Usage

  1. Enter your Facebook email and password in the logindetails.py file.
  2. Run the script using python tinder_bot.py.
